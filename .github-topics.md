## Как добавить Topics в GitHub репозиторий (ручной способ)

### Способ 1: Через веб-интерфейс GitHub

1. Откройте ваш репозиторий: https://github.com/semenovsd/mcp-claude-code
2. Найдите секцию **"About"** справа от названия репозитория (под кнопками "Code", "Issues", "Pull requests")
3. Нажмите на иконку **шестеренки (⚙️)** рядом с секцией "About"
4. В поле **"Topics"** введите следующие теги через запятую или добавляйте по одному:

```
mcp, mcp-server, model-context-protocol, claude, claude-code, claude-code-cli, ai, llm, ai-assistant, code-generation, cursor, cline, ide-integration, fastmcp, interactive, permissions, elicitation, session-resumption, progress-indication, automation, python
```

5. Нажмите **"Save changes"**

**Важно:** GitHub автоматически предложит варианты тегов при вводе. Выбирайте из предложенных или вводите новые.

### Способ 2: Через GitHub CLI (если установлен)

Если у вас установлен GitHub CLI (`gh`), выполните команду:

```bash
gh repo edit semenovsd/mcp-claude-code \
  --add-topic mcp \
  --add-topic mcp-server \
  --add-topic model-context-protocol \
```
