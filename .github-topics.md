## How to Add Topics

1. Go to your GitHub repository
2. Click on the gear icon (⚙️) next to "About" section
3. In the "Topics" field, add the topics listed above
4. Save changes

Or use GitHub CLI:
```bash
gh repo edit --add-topic mcp --add-topic mcp-server --add-topic model-context-protocol --add-topic claude --add-topic claude-code --add-topic claude-code-cli --add-topic ai --add-topic llm --add-topic ai-assistant --add-topic code-generation --add-topic cursor --add-topic cline --add-topic ide-integration --add-topic fastmcp --add-topic interactive --add-topic permissions --add-topic elicitation --add-topic session-resumption --add-topic progress-indication --add-topic automation --add-topic python
```
